# Agent Skills

Repositório público de skills para agentes de desenvolvimento como Codex, Claude Code e ferramentas compatíveis com `SKILL.md`.

## Skills disponíveis

### repo-health-audit

Analisa qualquer repositório de código fonte e gera relatório sobre:

- arquitetura
- dívidas técnicas
- observabilidade
- logs
- segurança
- testes
- CI/CD
- dependências
- riscos
- roadmap de evolução

## Instalação rápida

```bash
curl -fsSL https://raw.githubusercontent.com/SEU_USUARIO/skills/main/installers/install.sh | bash
```
